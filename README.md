목적

본 저장소는 다음 항목을 통합 관리하기 위한 Personal PMO Repository이다.

- 일상 업무 관리 (GTD)
- 프로젝트 관리
- 프로그램 관리
- 리스크 관리
- 기술 의사결정 관리
- 아키텍처 관리
- 주간/월간 보고 관리
- 지식 및 참고자료 관리

업무 상태

상태| 설명
Inbox| 수집된 업무
Next Action| 즉시 수행 가능한 업무
Waiting For| 타인 응답 대기
Project| 2개 이상의 액션이 필요한 업무
Someday| 향후 검토
Done| 완료
Archive| 종료
